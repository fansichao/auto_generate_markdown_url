# auto_generate_markdown_url

根据文件目录，自动生成 MarkDown URL 链接。

目录树，效果如下

```bash
----demo\
    |----A1\
    |    |----A11\
    |    |    |----demo110.txt
    |    |    |----readme.md
    |    |----demo11.txt
    |    |----demo12.txt
    |    |----readme.md
    |----A2\
    |----demo01.txt
    |----demo02.txt
    |----readme.md

```
